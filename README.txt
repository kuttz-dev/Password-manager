¡Precauciones!
- Si pierdes la contraseña maestra las demás serán irrecuperables.
- No borres los archivos de la aplicación (el archivo .exe, ni borres nada de la carpeta /media).
- No pierdas el archivo cuentas.db, pues es ahí donde se guardan todas tus contraseñas.

¿Que tan seguras son las contraseñas generadas?
Una contraseña con las opciones por defecto tardaría 3 millones de años en ser obtenida por un ordenador. Sin embargo esto tambien dependera de que la contraseña que utilizes para encriptarla sea del mismo nivel.

¿Cómo usar la aplicación?
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
#1. Pestaña "Panel principal"#
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1.1 Generar contraseña: genera una contraseña a partir de la configuración guardada en opciones.ini (las cuales se pueden editar en la pestaña configuración).
1.2 Icono de ojo: revela u oculta la contraseña.
1.3 Icono de documento: Copia lo que haya en la casilla de texto.
1.4 Servicio: Si ingresa una pagina web (ej: netflix.com) la aplicacion intentara guardar el favicon (icono de la pestaña) en la entrada de la db.
1.5 GUARDAR: Guarde todos los campos ingresados en "cuentas.db". Para realizar esta acción se solicitará una contraseña maestra, la cual se usará para encriptar cualquier contraseña ingresada. (NO OLVIDES ESTA CONTRASEÑA MAESTRA, O LA CONTRASEÑA ENCRIPTADA SERÁ IRRECUPERABLE)

~~~~~~~~~~~~~~~~~~~~~~~~~~~~
#2. Pestaña "Base de datos"#
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
2.1 En esta pestaña podrá ver todos los datos guardados en la base de datos.
2.2 Icono candado: revela u oculta las contraseñas encriptadas utilizando la contraseña maestra.
2.3 Utiliza la lista ("categoría", "usuario", "mail", "servicio") junto con la casilla de texto para establecer un filtro para buscar lo que desee en la base de datos.
2.4 Icono lupa: busque en la base de datos con el filtro establecido (la opción de la lista elegida y el texto de la casilla de texto).
2.5 Icono lápiz: alterne entre el modo edición. En este modo cualquier cambio que realices será guardado en la base de datos.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~
#3. Pestaña Configuración"#
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
3.1 Las opciones que se encuentren tildadas se usarán para definir las características de las contraseñas generadas.
3.2 La opción "fabricado con amor" no tiene ninguna funcionalidad.
3.3 La cantidad de caracteres está limitada para no perjudicar el funcionamiento de la aplicación.
3.4 El botón guardar re escribirá el archivo “opciones.ini” con cada opción marcada. Por favor solo edite este archivo a través de este panel.
3.5 Icono de pregunta: abrira un menu con información de contacto del creador y licencias de uso de los iconos.
